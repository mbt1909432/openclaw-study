# OpenClaw 安装指南

## 前提条件

OpenClaw 需要 Node.js 22 或更高版本。

### 检查 Node.js 版本

```
node --version
```

如果版本低于 22，请先升级 Node.js。

### 安装 Node.js

#### Windows 系统

1. 从 https://nodejs.org/ 下载
2. 选择 LTS 版本（包含 Node.js 22+）
3. 运行安装程序并按提示操作
4. 安装完成后重启终端

#### Linux/macOS（使用 nvm）

```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
source ~/.bashrc
nvm install 22
nvm use 22
```

#### Linux（使用 apt）

```
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt-get install -y nodejs
```

## 安装 OpenClaw

### 方法一：npm（推荐）

```
npm install -g openclaw
```

### 方法二：curl 脚本

```
curl -fsSL https://get.openclaw.ai | bash
```

### 验证安装

```
openclaw --version
openclaw --help
```

## 系统要求

| 资源 | 最低 | 推荐 |
|------|------|------|
| CPU | 1 核 | 2 核+ |
| 内存 | 1GB | 2GB+ |
| 磁盘 | 500MB | 2GB+ |

## 下一步

安装完成后，运行配置向导：

```
openclaw onboard --install-daemon
```

详见 /docs/quickstart 快速开始。
