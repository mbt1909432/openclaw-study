# OpenClaw 工作空间（Workspace）

## 什么是工作空间？

工作空间是 AI 机器人的"家"，就像你的房间一样，里面有：

- 机器人的"使用说明书"（告诉它怎么工作）
- 机器人的"记忆"（记住之前说过的话）
- 机器人的"工具箱"（各种技能）

## 目录结构

```
~/.openclaw/workspace/          # 默认工作空间
├── AGENTS.md                   # 机器人操作指令
├── SOUL.md                     # 人格和语气
├── USER.md                     # 用户信息
├── IDENTITY.md                 # 机器人身份
├── TOOLS.md                    # 工具说明
├── HEARTBEAT.md                # 心跳检查清单
├── BOOT.md                     # 启动检查清单
├── BOOTSTRAP.md                # 首次运行仪式
├── MEMORY.md                   # 长期记忆
├── memory/                     # 每日记日志
│   ├── 2024-01-15.md
│   └── 2024-01-16.md
├── skills/                     # 技能目录
└── canvas/                     # 画布文件
```

## 引导文件说明

这些文件会在每次对话开始时自动加载，告诉机器人怎么工作：

| 文件 | 用途 | 什么时候加载 |
|------|------|--------------|
| AGENTS.md | 操作规则、优先级 | 每次对话 |
| SOUL.md | 人格、语气、边界 | 每次对话 |
| USER.md | 用户是谁、怎么称呼 | 每次对话 |
| IDENTITY.md | 机器人名称、风格 | 每次对话 |
| TOOLS.md | 工具说明 | 每次对话 |

## 记忆文件

| 文件 | 用途 |
|------|------|
| MEMORY.md | 精选的长期记忆 |
| memory/YYYY-MM-DD.md | 每日记忆日志 |

## 如何编辑这些文件？

### 方法一：通过 Web 面板

1. 打开 http://127.0.0.1:18789/
2. 点击「Agent Files」
3. 选择要编辑的文件
4. 修改后保存

### 方法二：直接编辑文件

```
# 编辑人格文件
nano ~/.openclaw/workspace/SOUL.md

# 编辑用户信息
nano ~/.openclaw/workspace/USER.md
```

## 不在工作空间的文件

这些文件在 ~/.openclaw/ 目录下，不要放到工作空间：

| 路径 | 说明 |
|------|------|
| ~/.openclaw/openclaw.json | 配置文件 |
| ~/.openclaw/credentials/ | 凭证、API Key |
| ~/.openclaw/agents/ | 会话记录 |
