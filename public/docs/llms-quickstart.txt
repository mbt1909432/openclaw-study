# OpenClaw 快速开始

## 配置向导

OpenClaw 提供交互式配置向导来设置您的网关：

```
openclaw onboard --install-daemon
```

## 配置步骤

1. **安全警告** - 确认您了解安全影响
   ```
   确认安全警告？Yes/No
   ```

2. **配置模式** - 初学者选择 QuickStart
   ```
   选择模式：QuickStart / Advanced
   ```

3. **模型供应商** - 选择您的 AI 供应商
   - Anthropic（Claude）
   - OpenAI（GPT）
   - OpenRouter（多模型）
   - 其他供应商...

4. **消息平台** - 配置您的渠道
   - 飞书/Lark
   - Discord
   - Slack
   - Telegram

5. **技能** - 可选：启用托管技能

6. **钩子** - 可选：配置事件钩子

## 配置完成后

### 启动 Gateway

```
# 启动网关服务
openclaw gateway start

# 检查状态
openclaw status
```

### 访问 Web 面板

在浏览器中打开：

```
http://127.0.0.1:18789/
```

### 使用 TUI 终端界面

```
# 启动终端界面
openclaw tui

# 或使用简写
claw tui
```

## Web 面板功能

| 功能 | 说明 |
|------|------|
| 会话 | 查看和管理对话会话 |
| 定时任务 | 调度自动化任务 |
| Agent 文件 | 编辑 AGENTS.md、SOUL.md、USER.md 等 |
| 技能 | 管理已安装的技能 |
| 日志 | 实时日志查看器 |
